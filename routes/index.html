About This Project

This is an academic research prototype developed as part of a university project to explore AI-based traffic analysis. It demonstrates automated image classification and agentic workflow concepts in a controlled demo environment.

  • Type: Non-commercial academic research project
  • Purpose: Educational demonstration and research evaluation
  • Status: Prototype / Proof of concept
  • Audience: Researchers, educators, project reviewers, and authorized test users

How It Works

  • Users send traffic images via WhatsApp with prior opt-in consent
  • An AI pipeline analyzes the image and classifies potential violations
  • The system determines confidence levels and suggests verification requirements
  • Users receive an automated summary reply

Opt-In & User Consent

Use of this service requires explicit opt-in consent. By sending an image, users confirm:

  • They have provided their phone number voluntarily for this demo
  • They consent to receive automated replies from this research prototype
  • They understand this is an academic project with no legal authority
  • They may opt out at any time by blocking the number or requesting removal
